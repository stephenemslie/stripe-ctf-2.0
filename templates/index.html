# Container orchestrated Stripe CTF 2.0

Back in 2012, Stripe hosted a web security capture the flag game. Each level contained a security flaw that could be exploited to steal the password to the next level. When the game was finished, the servers were taken offline and the source code for was made availalbe on github.

This project brings it back to life by containerising each level with docker, orchestrating the services with docker-compose, and managing access and game state with a reverse proxy written in Go.

Enjoy!
